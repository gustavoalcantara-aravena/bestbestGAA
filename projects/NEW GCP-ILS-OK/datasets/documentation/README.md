# DIMACS Graph Instances

This dataset contains the DIMACS graph instances in ASCII graph format, along with source for the `binfmt` translator software for converting between ASCII and binary graph formats and the format [specification](./ccformat.pdf) in PDF.

## Fuentes de descarga

Este conjunto de datos fue descargado de las siguientes fuentes:

- **GitHub Repository**: [BartMassey/instances](https://github.com/BartMassey/instances) - Una copia limpia y bien mantenida de los DIMACS instances
- **Original Source**: [CMU TEPPER COLOR Instances](https://mat.tepper.cmu.edu/COLOR/instances.html) - La fuente original de los DIMACS graph instances
